Chapter 4 pull vs fetch

  • ‘fetch’ pobiera tylko meta dane ze zdalnego repo. To umożliwia np. zbadanie roznic miedzy repo lokalnym i zdalnym. Potem mozemy wykonac np. merge
  • ‘pull’ wykonuje ‘fetch’ i ‘merge’ jednoczesnie.
